Динамические системы

Динамические системы — математический язык описания эволюции систем во времени, общий для классической системологии, кибернетики и синергетики. В центре внимания — качественное поведение траекторий в фазовом пространстве: устойчивость, периодичность, хаос, переходы между режимами.

Классическая теория динамических систем (Пуанкаре, Ляпунов, Биркгоф) даёт понятийный аппарат: аттракторы и репеллеры, показатели Ляпунова, бифуркации. Бифуркация — качественное изменение поведения системы при плавном изменении параметра — является одним из ключевых механизмов возникновения новых режимов и структур.

Кибернетика использует динамические системы для описания систем управления: устойчивость замкнутого контура, частотные характеристики, передаточные функции. Понятие ультрастабильности Эшби описывает многоуровневые динамические системы, способные перестраивать свои параметры для сохранения устойчивости.

Синергетика (Хакен) вводит понятие параметров порядка и принцип подчинения: вблизи точки неустойчивости медленные моды подчиняют себе быстрые, и поведение сложной системы с многими степенями свободы описывается малым числом уравнений. Это делает динамические системы мостом между микроскопическим описанием и макроскопическими структурами.
