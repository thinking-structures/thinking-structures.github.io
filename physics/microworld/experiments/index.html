Эксперименты

Физика элементарных частиц — одна из немногих фундаментальных наук, где экспериментальные данные крупнейших установок становятся публично доступными. Ниже — подборка открытых ресурсов: от сырых данных коллайдеров до сводных таблиц свойств частиц. Каждый из них открывает возможность работать с реальными данными, а не только читать о результатах.

    CERN

    CERN Open Data Portal

    Открытые данные экспериментов LHC — ATLAS, CMS, LHCb, ALICE. Включает реальные данные столкновений, программное обеспечение для их анализа и учебные наборы данных. Объём измеряется петабайтами; для начала работы есть специально подготовленные облегчённые наборы и Jupyter-ноутбуки.

    opendata.cern.ch
    HEP

    HEPData

    Репозиторий числовых данных из опубликованных статей по физике высоких энергий. Когда статья выходит в журнале, авторы депонируют сюда таблицы, графики и данные в машиночитаемом формате. Позволяет воспроизводить результаты и сравнивать данные разных экспериментов напрямую.

    hepdata.net
    IceCube

    IceCube Neutrino Observatory

    Нейтринная обсерватория IceCube — детектор объёмом один кубический километр, расположенный в антарктическом льду на глубине до 2,5 км. Регистрирует нейтрино высоких энергий из астрофизических источников. Часть данных коллаборации доступна публично — отдельные релизы можно найти в разделе открытых данных на сайте проекта.

    icecube.wisc.edu/science/data-releases
